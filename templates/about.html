About

The purpose of this application is to predict the variety of an iris flower given its sepal width, sepal length, petal width and petal length. The data is from the famous iris dataset for machine learning

Whilst developing this application I had the following goals:-

  • To learn about Python Web Frameworks (I have used Flask for this application)
  • Showcasing Python for producing a data science product
  • Deploying the application on an Amazon EC2 instance

This application was produced using Flask, Scikit-Learn, MatplotLib, Seaborn and Bootstrap.

In a future version I would like to allow the user to choose the machine learning algorithm and parameters
